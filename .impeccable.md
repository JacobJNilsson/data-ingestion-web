# Design Context: CSV Contract Analyzer

## Target Audience
- **Primary**: B2B SaaS founders and developers building on Supabase/Postgres
- **Secondary**: Data engineers and technical decision-makers evaluating data ingestion tools
- **Context**: Discovery phase—users are evaluating whether this solves their "messy CSV import" problem

## Use Cases
1. **Demo & Discovery**: Upload a sample CSV to see instant contract analysis (structure, types, quality)
2. **Quick Validation**: Before committing to an integration, verify what the tool can detect
3. **Sales Conversations**: Show the tool's capabilities during customer discovery calls

## Brand Personality & Tone
- **Professional but approachable**: This is a serious developer tool, not a toy, but it should feel friendly and fast
- **Data-first**: The interface should feel precise, analytical, and trustworthy—users are trusting us with their data
- **Transparent**: Show exactly what we detected (no black boxes)—build confidence through clarity
- **Effortless**: Uploading a CSV and getting insights should feel instant and magical, not clunky

## Quality Bar
- **Phase 0 MVP**: Functional demo to validate demand in customer discovery conversations
- **Goal**: Polished enough to not embarrass us in sales calls, but not over-engineered
- **Ship fast**: Good enough to gather feedback, not pixel-perfect flagship

## Differentiation
- **Instant insights**: Most tools make you configure everything first—we analyze on upload
- **No AI slop aesthetic**: Avoid the cyan-on-dark, purple-blue gradient, glassmorphism look that screams "AI demo"
- **Data personality**: This is a tool for people who care about data—the design should reflect precision and clarity

## Current State
- Working Next.js app with CSV upload and contract display
- Currently using gradient backgrounds (blue/indigo/purple tones) which may lean toward "AI slop"
- Functional but described as "boring" by user
- User wants default light mode (not dark mode)

## Desired Direction
- More distinctive, memorable design
- Light mode by default
- Professional enough for B2B sales conversations
- Engaging enough to stand out in someone's memory
- Clear visual hierarchy showing the analysis results
